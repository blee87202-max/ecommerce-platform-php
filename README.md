# E-Commerce Platform (PHP & MySQL)

A full-featured local e-commerce web application built using PHP, MySQL, HTML, CSS, and JavaScript.

## Features
- User authentication & session management
- Product management (CRUD)
- Shopping cart system
- Order tracking
- Admin dashboard
- Role-based access control

## Technologies Used
- PHP (Core PHP)
- MySQL
- HTML5 & CSS3
- JavaScript (ES6)

## Architecture
- MVC Structure
- Normalized relational database
